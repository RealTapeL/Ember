# 知识传承工坊 — 设计PRD

## 项目定位

一个面向毕业生的知识传承工具。用户（学长学姐）上传自己的学习文档、笔记或论文，系统将其转化为结构清晰、交互友好的教程，供学弟学妹们按步骤学习。这是一个从「个人知识」到「可传承教程」的炼金过程。

## 情感基调

温暖、学术、仪式感。不是冷冰冰的AI转录工具，而是一个知识传递的桥梁。视觉语言上融合「手写信的温度」与「数字时代的清晰」。

## 视觉体系

### 色彩

以**深靛蓝**（#1e293b）为主背景，象征深夜图书馆的宁静与专注；**暖琥珀**（#f59e0b）作为强调色，代表知识的温度和光芒；**米白**（#fef3c7）用于文字和高光，营造旧纸张的温润感。

### 字体

中文标题使用「霞鹜文楷」（lxgwwenkai），这是一款开源的中文楷体，兼具书法韵味与现代可读性；正文使用系统无衬线字体栈（PingFang SC, Microsoft YaHei, sans-serif），确保长文阅读的舒适度。

### 间距

采用宽松的纵向节奏（section padding: 80-120px），让内容有呼吸感。每个功能区块之间用极细的渐变分割线过渡，避免生硬的边界。

## 页面结构

### 首页（Landing）

一个单页应用，包含以下纵向排列的区块：

#### Hero 区

全屏高度，深靛蓝背景。中央是一句手写风格的标语：**"把你的笔记，变成他们的阶梯"**。标语下方有一个微妙的粒子动画——数百个金色光点像萤火虫一样在黑暗中缓慢漂浮，当鼠标靠近时，光点会温和地散开。底部有一个向下滚动的指示箭头，伴随呼吸动画。

#### 上传区

这是核心交互区。一个宽大的拖拽上传区域，边框是虚线渐变（靛蓝到琥珀色）。未激活时显示：「拖拽你的文档到这里，或者点击选择文件」，配有一个文档图标。拖拽进入时，边框发光（琥珀色阴影脉冲），文字变为「松开即可开始转化」。上传后显示进度——一个圆形进度环，中央显示百分比，环的颜色从靛蓝渐变为琥珀色。

#### 转化展示区

上传完成后，页面平滑滚动到此区域。这里以**分屏对比**的形式展示效果：左侧是原始文档的缩略预览（暗化处理，像被封存的记忆），右侧是生成的教程结构——以**步骤卡片**的形式纵向排列。卡片采用**3D倾斜效果**：鼠标在卡片上移动时，卡片会跟随鼠标方向产生轻微的3D旋转（perspective: 1000px, rotateX/Y ±5°），仿佛卡片悬浮在空中。每张卡片包含步骤序号、标题和简短描述。

#### 预览区

点击任意步骤卡片后，右侧（或下方）展开该步骤的详细内容。这是一个**富文本阅读区**，支持代码高亮、重点标注、交互式问答。内容呈现采用**打字机效果**——文字逐字出现，模拟学长学姐在亲自讲解的感觉。打字速度适中（每字符30ms），不会让人等待太久，但保留了"正在书写"的温度。

#### 传承走廊（装饰区）

页面底部是一个水平滚动的视觉画廊。展示「传承」的隐喻图像——手递手、灯塔、接力棒等意象，以**无限滚动**的方式从右向左缓慢移动。每张图像是黑白处理，hover 时恢复色彩并放大1.05倍。这个区域不承载核心功能，营造氛围。

## 核心交互

### 文档上传

- 支持格式：PDF、Word、Markdown、TXT
- 拖拽上传：dragenter 时边框发光，drop 时触发上传
- 上传动画：圆形进度环（SVG stroke-dasharray 动画）
- 完成后自动滚动到转化展示区

### 教程生成

- 前端模拟解析过程（因为后端需要AI处理，这里设计前端体验）
- 解析进度以**步骤条**展示：提取文字 → 理解结构 → 生成步骤 → 优化排版
- 每个步骤有独立的图标和完成动画（对勾缩放弹出）

### 3D 步骤卡片

- 容器设置 `perspective: 1000px`
- 卡片根据鼠标位置计算 `rotateX` 和 `rotateY`（最大±5°）
- 叠加一层光泽渐变（linear-gradient），随鼠标移动改变角度
- 过渡使用 `transform 0.1s ease-out`，确保跟手但不抖动

### 打字机效果

- 逐字显示，每字符30ms
- 遇到代码块时暂停，整块显示
- 支持「加速」按钮（点击后速度提升至每字符5ms）

## 动画规划

| 动画效果 | 实现方式 | 复杂度 |
|---------|---------|-------|
| Hero 粒子光点 | Canvas 2D，100个粒子，随机速度，鼠标排斥 | 高 |
| 上传进度环 | SVG + CSS animation，stroke-dashoffset | 中 |
| 3D 倾斜卡片 | React + onMouseMove + CSS transform | 中 |
| 打字机效果 | setInterval + 字符切片 | 中 |
| 步骤条完成动画 | CSS keyframes，scale + opacity | 低 |
| 传承走廊滚动 | CSS animation，translateX infinite | 低 |
| 页面滚动过渡 | CSS scroll-behavior: smooth | 低 |

## 技术栈

- React 18 + TypeScript + Vite
- Tailwind CSS
- shadcn/ui（用于按钮、输入框等基础组件）
- marked（Markdown 渲染）
- highlight.js（代码高亮）

## 需要生成的图片

1. **Hero 背景纹理** — 深夜图书馆的模糊背景，暖色台灯灯光，1张
2. **传承走廊图片** — 手递书本、灯塔、阶梯等意象，4-5张
3. **空状态插画** — 一个打开的笔记本和羽毛笔，1张（用于无文档时的默认展示）

## 用户旅程

1. 打开页面 → 看到 Hero 标语和萤火虫粒子 → 被氛围吸引
2. 滚动到上传区 → 拖拽或选择自己的学习文档
3. 观看上传进度 → 自动进入转化展示区
4. 看到文档被解析为步骤卡片 → 鼠标悬停感受3D效果
5. 点击卡片 → 观看打字机效果呈现详细内容
6. （想象）学弟学妹通过这个链接访问 → 按步骤学习

## 核心信息

这不是一个工具，而是一次**知识的传递**。每一次上传，都是学长学姐对学弟学妹的隔空喊话："这条路我走过，现在我把经验留给你。"
